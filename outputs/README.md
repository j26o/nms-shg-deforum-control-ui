# Local Outputs

This folder is the project-local home for generated prototype outputs that are not committed to Git.

- `logs/`: frontend dev-server logs, Playwright/manual run logs, and other local execution logs.
- `previews/`: lightweight mock or exported preview artifacts produced by the workbench.
- `exports/`: reviewed preset JSON, readable reports, and handoff exports.

Automatic1111 Deforum outputs are kept inside the project-local runtime at:

```text
render-tools/stable-diffusion-webui/outputs/
```

Do not move generated logs, videos, frames, settings files, checkpoints, or temporary render artifacts to external project folders.
